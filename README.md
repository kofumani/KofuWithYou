# Kofu with You : Game Story

30代の男性ゲーム実況者「こふねゐ」と一緒に過ごす、ローカル完結のチル作業Webアプリです。

## できること

- 「作業を開始する。」を押して、一緒に作業した実時間を記録
- 累計作業時間に応じて、こふねゐとの仲良し度が上昇
- 本人のツイートを基にした指定の25セリフをランダム表示
- 画面右下のこふねゐをクリックすると `mid.mp3` のボイスを再生
- 初期起動時に画像・BGM・ボイスを読み込む専用プログレス画面を表示
- 全アセットの準備完了後、「スタート」を押してメイン画面へ移動
- 読み込み失敗時はメイン画面を開かず、専用の再読み込みボタンを表示
- 数分ごとに `perippa.webp` が背景を横切るランダム飛行演出
- 甲府をテーマにした4種類の背景をカスタマイズ
- `thumnail/` の全画像から動的に生成される「こふねゐ ギャラリー」
- `old.png` の輪郭・色・重心を解析し、画像ごとに `kofu.webp` を自動配置
- 拡大画面でこふねゐの中央をドラッグして移動、四隅をドラッグしてサイズ変更
- ワンタッチでこふねゐを左右反転
- 編集内容をブラウザへ保存し、原寸の合成済みPNGをダウンロード
- `lofi.mp3`、雨音、夜の虫の音量を個別に調整
- 記録、背景、サウンド設定をブラウザへ自動保存

## 起動

`index.html` を直接開くか、このフォルダで次を実行してください。

```sh
python3 -m http.server 8765
```

その後、`http://localhost:8765` を開きます。

## キーボード操作

- `Space`: 作業を開始 / 終了
- `M`: BGMを再生 / 一時停止
- `Esc`: 開いている設定画面・ギャラリーを閉じる
- `←` / `→`: ギャラリー拡大表示で前後の画像へ移動

BGMと環境音はブラウザの自動再生制限により、最初の操作後から再生できます。

## ギャラリー画像の更新

`thumnail/` に画像を追加した後、次を実行すると配置解析・軽量プレビュー・一覧データが更新されます。

```sh
python3 scripts/build_gallery_manifest.py
```

解析にはImageMagickの `magick` コマンドを使用します。ファイル名に日本語、空白、`#` が含まれていてもURL安全なパスへ変換されます。
